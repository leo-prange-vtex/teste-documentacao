# Documentação de Referência

Esta pasta contém os PRDs (Product Requirements Documents) do projeto.

## 📄 Documentos

- **prd-001.md**: Sistema de Gerenciamento de Produtos
- **prd-002.md**: Plataforma de Analytics em Tempo Real

## 🔄 Como Atualizar

1. Crie uma branch com um nome descritivo
2. Edite os arquivos .md
3. Faça commit e push
4. Abra um Pull Request
5. Após o merge, a sincronização com Google Docs acontecerá automaticamente

## ⚙️ Processo de Revisão

- Code Review é obrigatório
- Cada PR deve ter pelo menos 2 aprovações
- Testes automáticos devem passar

## 📱 Suporte

Para dúvidas sobre a sincronização, consulte o README principal.
